# Laravel Service Booking System

A multi-vendor service booking system built with Laravel.

## Features
- Role-based authentication (Admin, Vendor, Customer)
- Vendor service management
- Time-slot based booking
- Booking status management
- Clean MVC architecture

## Tech Stack
- Laravel
- Blade
- MySQL

## Use Case
Ideal for salons, repair services, consultants, clinics, and freelancers.
